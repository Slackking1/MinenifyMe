This is a simple proff of concept plugin for grasshopper. 

Currently the plugin provide a fast method of splitting curves and surfaces into geometry to be used with minecraft functions in datapacks. 
You can do all the thing in this plugin with native gh components, this just make things faster and cleaner. 


Minecraft list from this repo, current version of list 18.9 https://github.com/Radvylf/minecraft-lists
